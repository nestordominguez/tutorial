Instalación

Instalar react native

$ npm install -g react-native-cli

Ahora hacemos los siguiente pásos

  1. Vamos con la consola hasta la carpeta en donde vamos a crear el proyecto
  2. Creamos la aplicación react:
    Crear react native app
    $ react-native init NombreDeApp

    Esto crea una carpeta con el nombre NombreDeApp

  3. Ingresamos a la carpeta NombreDeApp:
    Ingresar en la carpeta NombreDeApp
    $ cd NombreDeApp
  4. En caso de no tener instalado rnpm hacemos este paso, si no pasamos alsiguiente
    Instalar rnpm
    $ npm install rnpm -g
  5. Utilice RNPM de instalar y conectar el Facebook SDK para paquetes React Native y sus dependencias.
    Instalar dependencias
    $ rnpm install react-native-fbsdk

    Estas dependencias permiten conectar react native con un emulador
